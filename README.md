# Локальный мобильный чат (VPoTOKE) (WIP)

## Описание
VPoTOKE - мобильное приложение, разработанное в рамках учебной практики.
Представляет собой локальный клиентский чат (мессенджер) для обмена текстовыми сообщениями и файлами в режиме реального времени с использованием локального сервера и системой аутентификации.

# Особенности приложения
- Общий групповой чат
- Регистрация и авторизация через Firebase Authentication (будет добавлена в следующей версии)
- Отправка и получение файлов (хранятся на локальном сервере)
- Настройка интерфейса приложения
- Управление уведомлениями и вибрацией
- Настройки профиля пользователя

## Функциональные требования (выполнены)
- Регистрация;
- Авторизация;
- Отправка сообщений;
- Получение сообщения;
- Просмотр истории переписки;
- Изменение профиля пользователя (редактирование профиля);
- Включение / выключение уведомления;
- Включение / выключение вибрации;
- Просмотр информации об приложении;
- Настройка внешнего вида (настройка цвета, размера текста, стиля чата).

# Установка и запуск
## Системные требования

### Для клиента

- Система Android 8.0 и выше
- ОЗУ: 2 ГБ
- Сеть: Наличие модуля Wi-Fi или сотовой связи
- Хранилище: 50 МБ свободного места

### Для разработки

- IDE Android Studio
- Android SDK версии 26 и выше
- Аккаунт Firebase для аутентификации
- Локальный сервер Windows Server для хранения сообщений


## Запуск приложения

### Для пользователей

1. Скачайте последнюю версию приложения из раздела `Releases`
2. Установите приложение
3. После установки вы можете запустить его

### Для разработчиков

1. Скачайте исходный код приложения
2. Загрузите его в Android Studio
3. Для локального сервера вам нужно иметь устройство с Windows Server или Linux
4. Настройте IP-адреса и порты вашего сервера и вставьте их в поля кода
5. Соберите приложение из исходников
6. После появлении apk файла вы можете запустить приложение в эмуляторе или на реальном устройстве
